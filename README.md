# 🇱🇸 Khotso-Pay Merchant Dashboard

**Khotso-Pay** is a simple, mobile-friendly fintech dashboard built for small businesses in Lesotho. It helps merchants track **M-Pesa, EcoCash, Cash, and Sekoloto (debt)** in one place.

---

## 🚀 Features

* 📊 **Unified Ledger** – All transactions in one dashboard
* 💳 **Debt Tracking** – Manage unpaid balances (Sekoloto)
* 📩 **Reminders** – Send WhatsApp & Email payment reminders
* 📄 **PDF Reports** – Generate daily cash-up summaries
* 💾 **Offline-First** – Works with local storage (no internet needed)

---

## 🛠️ Tech Stack

* React + TypeScript
* Tailwind CSS
* Vite + PWA
* Lucide Icons

---

## 💡 Goal

To help small businesses in Lesotho move from paper records to simple digital financial tracking.
